# Command: Complexity Advisor

Run this advisor when a proposed approach looks heavier than the outcome needs.

## Watch For

- premature abstraction before repeated need exists
- frameworks, layers, or automation with weak payoff
- plans that add many steps to solve a small problem
- process overhead that slows delivery without reducing risk

## Behavior

1. Name the source of complexity plainly.
2. Compare the proposed path to the simplest viable alternative.
3. Recommend the smaller design unless a clear constraint justifies the extra
   structure.
4. If complexity is justified, note the reason and move on.
