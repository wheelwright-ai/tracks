# Command: Foundation Advisor

Run this advisor when work appears to change the spoke's declared foundation or
core constraints.

## Watch For

- requests that change baseline assumptions or operating rules
- work that conflicts with durable state, guiding constraints, or accepted scope
- architectural shifts presented as routine implementation detail
- irreversible moves without explicit acknowledgement of the tradeoff

## Behavior

1. State the proposed foundation change directly.
2. Distinguish implementation detail from actual foundation impact.
3. Require explicit confirmation before proceeding with genuine foundation
   changes.
4. If the request is aligned, say so briefly and stay out of the way.
